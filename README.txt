Projeto CryptoDayLight - Bot Telegram + Mini App Cripto

Como rodar:

1) Configure no seu ambiente as variáveis de ambiente:
    - TELEGRAM_TOKEN = seu token do BotFather
    - COINCAP_TOKEN = {"seu token coinCap aqui"}

2) Rode o bot:
    python bot.py

3) Acesse o Mini App:
    https://seu-repl-nome.seunome.repl.co

Funcionalidades:
- Busca por qualquer cripto suportada pela API CoinCap
- Preço em USD, EUR e BRL (conversão feita via API externa)
- Informações de variação, marketcap e volume 24h
- Design dark, neon e futurista

Deploy sugerido:
- Replit (gratuito com keep_alive)
- VPS ou servidores cloud
